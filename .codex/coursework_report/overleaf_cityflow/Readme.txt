README
--------------------------

To use latex for your reports:

https://www.overleaf.com/ is an online LaTeX editor. I don't mind about which template you use.

If you use this template, modify only the GamesEngineeringTemplate.tex file and the GamesEngineering.bib file. Leave Config.tex alone, as it sets up the editor to display the document in the correct format.

References go in the GamesEngineering.bib file, and references in the correct BibTex format can be found on Google Scholar for most things.

Then you are ready to go!

Stackoverflow is your friend when you have problems!
